Introduction

Realtime, open source workflows execution web application.

Tideflow integrates with:

  • Tideflow's agent tool
  • Github Webhooks
  • RSS Channels
  • Inbound Endpoints
  • Cron Based Triggers
  • PDF Creation
  • Email Sending
  • Webforms
  • Web scraping
  • many more to come

Quick Start

Make sure you have MetorJS installed.

Then run:

cd tideflow
  meteor
